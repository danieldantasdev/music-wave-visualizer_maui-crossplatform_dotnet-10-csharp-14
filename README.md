# music-wave-visualizer_maui-crossplatform_dotnet-10-csharp-14
Projeto experimental de visualização gráfica de áudio em tempo quase real, desenvolvido com .NET MAUI (.NET 10), com foco em computação gráfica e processamento digital de sinais. Permite upload de arquivos WAV, reprodução de áudio e renderização dinâmica da forma de onda com SkiaSharp.
